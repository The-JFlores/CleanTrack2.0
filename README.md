# CleanTrack 2.0

CleanTrack 2 is a cleaning management app built with SwiftUI and Firebase.  
Its main purpose is to track cleaning schedules for 10 buildings, storing key information such as:

- Alarm code  
- Address  
- Client contact  
- Email  
- Closing time  
- Cleaning frequency (1x or 3x per week)  
- Notification 30 minutes after closing  
- Real-time database sync  
- User authentication (Login / Register)  
- Daily schedule showing only todayâ€™s buildings
